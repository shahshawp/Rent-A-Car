Enhancements Description
Responsive Design Responsive Design is actually a design which is based on the screen resolutions of mobile phone. By using "@media screen and (max-width:"resolutions of phone", you can change the CSS codes at different resolutions to make it Responsive.(Try to scale the size of browser to show Responsive) extra responsive css file is required.
Font Face Font Face is a CSS code that can make a font which is not available in your computer to show on you website. It can define a "non-local" font as a "local" font by using "@fon-face", and the font can be used everywhere(Default font of whole website is Arial, san-serif)
Filter Filter can change the content's visual effects, like blur, brightness or make it change into black and white
Box Shadow Box Shadow can make box to have a shadow.(hover to the car box)
Favicon Favicon is an web tab icon, using "link rel="shortcut icon"" tag can change the tab icon of your website(Please look at the tab icon)
